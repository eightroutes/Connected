## 당신의 동네, 당신의 세상

**CONNECTED**는 당신의 일상을 특별하게 만들어줄 동네 친구를 찾아주는 소셜 네트워킹 앱입니다.

---

### 🌟 주요 기능

- **위치 기반 매칭**: 가까운 곳에 있는 흥미로운 사람들을 만나보세요.
- **성격 분석**: MBTI와 관심사를 통한 매칭으로 당신과 잘 맞는 친구를 찾아드립니다.
- **취향 공유**: 음악, 영화 등 당신의 취향을 공유하고 비슷한 관심사를 가진 친구를 발견하세요.

---

### 🛠 기술 스택

- Swift & SwiftUI
- Firebase
- MapKit
- CoreLocation
